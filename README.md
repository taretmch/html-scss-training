# html-scss-training

## インストール

```sh
// node がインストールされていない場合
% brew install nvm
% nvm install v8.17.0
% node -v

// yarn がインストールされていない場合
% brew install yarn

// webpack がインストールされていない場合
% brew install webpack

// node のライブラリをインストールする
% yarn install

// scss をビルドする
% yarn run build

// scss のビルドを watch する
% yarn run build.watch
```
